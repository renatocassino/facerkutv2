# Facerkutv2

## Endpoints

```
/api/v2/communities -> List all communities
/api/v2/communities/:slug/im-in -> Check if user are a member of a community
```

## Tables

```
communities
users
topics
communities_users -> Members
```

## Commands

```bash
make build
make start
```

To run queries in terminal, you must run first

`import Ecto.Query`
